# Mono

A real-time, room-based sharing workspace for text and files.

## Features

- **Real-time Text Editing:** Collaborative text pad with instant synchronization.
- **File Sharing:** Upload and share images, videos, audio, and documents.
- **Room-based Collaboration:** Join or create rooms using simple codes.
- **File Management:** Download, rename, and delete shared files.
- **Responsive Design:** Works on desktop and mobile devices.

## Tech Stack

- **Frontend:** React, Tailwind CSS, Framer Motion, Lucide React
- **Backend:** Express, Socket.IO
- **Database & Storage:** Firebase (Firestore, Storage, Auth)

## Getting Started

1. Enter a room code on the landing page.
2. Start typing in the editor or upload files to the feed.
3. Share the room code with others to collaborate in real-time.
